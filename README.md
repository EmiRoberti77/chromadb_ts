## Chroma DB - node.js

install database first

```bash
python3 -m venv chroma_env
source chroma_env/bin/activate
pip install chromadb
chroma run
```

# set up node project

```bash
npm init -yes
npm i -D @types/node typescript ts-node
npm i chromadb chromadb-default-embed
```
